# DepthBook

iOSのデプス（深度）の解説書「[Depth in Depth](https://booth.pm/ja/items/1313752)」のサンプルコードです。

## 書籍の概要

iOSにおける「デプス」（深度）の取り扱いについて、基礎から応用まで詳しく解説した書籍です。

A5版、44ページ。iOS 12, Swift 5, Xcode 10対応。
